# Invoke-PlmJarBuilder

## SYNOPSIS
Invokes the PLM-Jar-Builder.

## SYNTAX

```
Invoke-PlmJarBuilder [-Offline]
```

## DESCRIPTION
The "Invoke-PlmJarBuilder" cmdlet leads through the execution of possible PLM-Jar-Buidler commands.

## EXAMPLES

### -------------------------- BEISPIEL 1 --------------------------
```
Invoke-PlmJarBuilder
```
